# FinanceAI - Smart Money Manager

AI-powered financial management PWA. Track income, expenses, and get smart budget insights.

## Features

- **Dashboard** — Overview with charts (income vs expenses, category breakdown)
- **Income Tracking** — Record monthly salary and earnings
- **Expense Tracking** — Categorize expenses (10+ categories)
- **Transaction History** — Search, filter, export to CSV
- **AI Insights** — Smart spending analysis and budget recommendations
- **Settings** — Theme (light/dark/system), currency selector (12 currencies), data backup
- **PWA** — Installable on mobile, offline support, home screen shortcuts
- **Responsive** — Works on mobile, tablet, and desktop
- **Dark Mode** — Full dark mode support

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **TailwindCSS**
- **Recharts** (charts)
- **Lucide React** (icons)
- **localStorage** (data persistence)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Install as Mobile App

Deploy to Vercel/Netlify, then open the URL on your phone and tap "Add to Home Screen".

## Currency Support

PKR, USD, EUR, GBP, INR, AED, SAR, CAD, AUD, CNY, JPY, TRY
